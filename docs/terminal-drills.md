# Terminal Drill – Day 1

Commands Practiced:

1. pwd – Print working directory
2. ls – List files
3. tree – Show directory structure
4. mkdir – Create directory
5. mv – Rename/move file
6. rm – Remove file/directory
7. git init – Initialize repo
8. git add – Stage changes
9. git commit – Commit changes
10. git push – Push to remote

Example Output:
- Verified repo structure using tree
- Verified remote using git remote -v
